Bem-vindo à página de links do FabLab


Esta página foi criada no intuito de ser um guia de arquivos, softwares e repositórios utilizados no FabLab para cada uma das máquinas.


Controle e Organização de Projetos

  • ClickUp

  • Programa online de organização de projetos e atribuição de tarefas.
  • Link do ClickUp

Circuitos Impressos

  • EasyEDA

  • Software utilizado para criação de placas de circuito impresso.
  • Link do EasyEDA

  • FlatCAM

  • Software que configura os parâmetros do arquivo utilizado na fresadora e converte o arquivo para G-Code.
  • Link do FlatCAM

  • Candle

  • Software utilizado para controlar a fresadora.
  • Link do Candle

  • Universal G-Code Sender (UGS)

  • Software utilizado para controlar a fresadora.
  • Link do UGS

Máquina de Corte a Laser

  • MakerCase

  • Site utilizado para fazer o design de caixas em MDF.
  • Link do MakerCase

  • CorelDraw/Inkscape

  • Softwares de desenho com suporte para arquivos vetorizados (svg) utilizados na cortadora a laser.
  • Link do Inkscape

  • k40

  • Software utilizado para conectar o computador à máquina de corte a lazer.
  • Link do k40

Programação Arduino

  • ArduinoIDE

  • Programa oficial da Arduino, utilizado para criar, revisar e enviar código para o controlador arduino
  • Link do ArduinoIDE

  • Tinkercad

  • Programa de simulação utilizado para simular circuitos eletrônicos utilizando arduino e demais componentes eletrônicos.
  • Link do Tinkercad

Impressora 3D

  • ThingVerse

  • Repositório de arquivos 3D para impressão
  • Link do Thingverse

  • UltimakerCura

  • Software utilizado para tratar os arquivos antes de manda-los para a impressora 3D
  • Link do Ultimaker Cura

  • SolidWorks/Fusion360

  • Software utilizado para criar modelos em 3D
  • Link do Fusion360
